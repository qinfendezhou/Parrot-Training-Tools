# Parrot-Training-Tools
"Parrot Training Tools" is an app that helps parrot lovers to train parrots. Users can learn about the information and training methods related to training the parrot here. It also has a powerful recording function to record what the user wants the parrot to say, or to record the moment the parrot speaks, which will be saved as a souvenir. Using this app, users can train their own parrots faster, and hope that every parrot lover can reap their own happiness.
